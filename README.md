# Simple Wallet

A simple Ethereum wallet frontend built with HTML, CSS, and JavaScript.  
It lets you connect MetaMask, view your wallet balance, estimate gas fees, send ETH, and view transaction history.

## Features

- Connect to MetaMask
- View wallet address and balance
- Estimate gas fees before sending
- Send ETH to another wallet address
- View local transaction history for the connected account
- Copy wallet address to clipboard

## Prerequisites

- A browser with MetaMask installed
- A MetaMask wallet
- Access to a test network such as Sepolia

## Setup

1. Install MetaMask in your browser.
2. Create or import a wallet.
3. Switch MetaMask to the Sepolia test network.
4. Get some Sepolia test ETH from a faucet.
5. Open the project in a local web server.
